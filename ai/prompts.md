# AI prompts and usage notes

AI was used during planning and implementation.

## Prompt 1

Describe the requirements from the take-home assignment and propose a Nuxt project structure.

## Prompt 2

Create a compact Nuxt SSR device catalog architecture with shared Zod schemas, Nitro API routes, server-side filtering, sorting and pagination.

## Prompt 3

Help implement catalog UI components using SCSS modules, typed props, responsive layout, theme tokens and accessible states.

## Prompt 4

Generate local `.webp` placeholder product images matching the JSON file paths.

## Prompt 5

Review UI issues such as loading flicker, scrollbar layout shift, select styling, dark theme contrast and product image sizing.

## Human review

All generated snippets were reviewed, adapted and integrated manually. Visual choices, architecture decisions and final trade-offs were made by the developer.
